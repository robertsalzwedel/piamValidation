# Print a summary of the input data

Print multiple metrics of all data objects given to
“validateScenarios()“ which might be helpful to spot and fix data
inconsistencies.

## Usage

``` r
showInputSummary(scen, hist, config)
```

## Arguments

- scen:

  scenario data as used in “validateScenarios()“

- hist:

  historical/reference data as used in “validateScenarios()“

- config:

  config as used in “validateScenarios()“
